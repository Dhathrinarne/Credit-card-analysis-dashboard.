## 📌 Project Overview

This project analyzes **credit card default behavior** using demographic and financial factors.

The goal is to identify patterns and insights that explain **who is more likely to default** and why.

## 📊 Dataset Description
The dataset includes the following variables:

* **Dependent Variable**

  * Default Payment (Yes/No)

* **Independent Variables**

  * Age
  * Gender
  * Education Level
  * Marital Status
  * Employment Status
  * Number of Kids
  * Ethnicity
  * Credit Limit & Billing Info

## 🔍 Key Insights

### 👩 Gender & Employment

* Females show slightly higher default counts than males
* Both employed and unemployed individuals have similar default trends

### 👨‍👩‍👧 Marital Status & Kids

* Married individuals have the highest defaults
* Customers with **0 kids dominate defaults**

### 🌍 Race & Demographics

* Caucasian group shows highest defaults
* Hispanic group shows lowest

### 🎓 Education

* University-educated individuals have the highest defaults
* Lower education levels show fewer defaults

### 🎂 Age Trend

* Default rate is highest between **25–35 years**
* Drops significantly after age 50

## 📈 Conclusion

* Younger individuals are more likely to default
* Education and marital status strongly influence credit behavior
* Employment alone is not a strong predictor
